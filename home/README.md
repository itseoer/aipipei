
# 首页模块

这个文件夹包含网站首页相关的所有代码文件。

## 目录结构

home/
├── components/ # 组件文件
├── styles/ # 样式文件
├── assets/ # 资源文件(图片等)
├── utils/ # 工具函数
└── README.md # 说明文档
